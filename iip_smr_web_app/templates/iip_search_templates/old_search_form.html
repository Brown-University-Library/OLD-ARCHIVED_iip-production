Search tips

You can select multiple values in a single field; these search linked as "OR"s (i.e., Greek OR Latin, Caesarea OR Ben Shemen). Between fields, multiple selections are linked by "AND," i.e., Greek AND Jewish AND in the Galilee; epitaph AND pagan. To de-select, highlight and press the Apple key (for Mac) or Ctrl (for PC). While this site allows for typical, but powerful, scholarly searches -- e.g., "all Christian inscriptions written in Latin from Jerusalem that might date from the fifth century" -- it also allows for inventive and quirky searches that reconfigure these inscriptions to open new windows into ancient society, culture and religion.
